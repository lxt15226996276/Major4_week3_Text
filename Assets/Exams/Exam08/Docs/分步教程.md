### 2.2 布局改造 · `LoginFormColumn`（**§11.25 U3/U7 · VLG 七步 · 必做**）

> Exam06 钮行 **HLG 七步同类** · 登录表单 **VLG** · **增删控件零手调 Pos**。  
> **U7 = 只挂 VLG、不 CSF、不 Reset Anchor、100×100 列盒** → 表单 **偏右/偏上**，与第六套 **U2** 同一类错。

| # | 操作 | 禁止 |
|:-:|------|------|
| 1 | 完成 **删除清单**（见 2.4） | — |
| 2 | LoginPanel 下 **Create Empty** → 命名 **`LoginFormColumn`**（**全拼写**） | `LoginFormColum` |
| 3 | **RectTransform**：Anchor **(0.5,0.5)(0.5,0.5)** · Pivot **0.5** · Pos **(0,0)** | 默认 **100×100** 小盒 |
| 4 | 加 **Vertical Layout Group + Content Size Fitter**（**两个都要**） | 只挂 VLG |
| 5 | 拖入 **AccountInput · PasswordInput · BtnLogin**（**先不要 Name**） | 拖完只改 Pos Y |
| 6 | 各子物体加 **Layout Element** Preferred **900×80**（Btn **200×60**） | — |
| 7 | **每个子物体** RectTransform：Anchor **Middle Center (0.5,0.5)** 或 **Top-Stretch 宽** · **Pos (0,0)** · 清掉 Prefab **Y=120/-35/180** 残留 | **Top-Left** · **Pos X≈450** |

#### `LoginFormColumn` · Vertical Layout Group

| 字段 | 设置值 | 含义 |
|------|--------|------|
| Padding | **10** | 内边距 |
| Spacing | **24** | 与 Exam06 钮行一致 |
| Child Alignment | **Middle Center** ★ | **禁止 Upper Left（默认 0）** |
| Control Child Size | W **关** · H **关** | 不拉变形 |
| Child Force Expand | W **关** · H **关** | 开=Input 被拉高 |

#### `LoginFormColumn` · Content Size Fitter ★

| 字段 | 设置值 | 含义 |
|------|--------|------|
| Horizontal Fit | **Preferred Size** | 列宽随 Input |
| Vertical Fit | **Preferred Size** ★ | 加 NameInput **自动增高** · **无 CSF = U7** |

#### `LoginFormColumn` · RectTransform

| 字段 | 设置值 | 含义 |
|------|--------|------|
| Anchor | **(0.5, 0.5) (0.5, 0.5)** | 表单块在 LoginPanel **居中** |
| Pivot | **(0.5, 0.5)** ★ | 增删 Input **上下对称扩展** |
| Pos | **(0, 0)** | 整组定位 · **不逐个 Input 写 Y** |
| Size Delta | **由 CSF 撑开** | **禁止**手填 **100×100** |

#### 每个 Input / BtnLogin · Layout Element + Anchor Reset

| 物体 | Layout Element | RectTransform（VLG 子级） |
|------|----------------|---------------------------|
| AccountInput | Preferred **900×80** | Anchor **(0.5,0.5)** · Pos **(0,0)** |
| PasswordInput | Preferred **900×80** | 同上 · **Hierarchy 第 2** |
| NameInput（2.3 加） | Preferred **900×80** | 同上 · **Hierarchy 第 3** |
| BtnLogin | Preferred **200×60** | 同上 · **Hierarchy 第 4** |

**Reset 操作（Unity）**：选中子物体 → RectTransform 右键 **Reset** 或 Anchor Preset **Middle Center** 按住 **Alt** 应用 → 再设 **LE** · **Pos 应为 (0,0)** · Inspector 仍显示 **driven by VerticalLayoutGroup** 是正常的。
