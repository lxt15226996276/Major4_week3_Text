## 断网考试自检（每套做完必做）

1. 断开网络，只开 Unity + 录屏软件  
2. **File → Build Settings** 确认 Scenes In Build 顺序正确  
3. Play 全流程走通  
4. Console 无红错  
5. 关 Unity 重开，无 Missing 引用
