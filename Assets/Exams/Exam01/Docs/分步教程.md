```csharp
using UnityEngine;
using UnityEngine.UI;
using UnityEngine.SceneManagement;

public class LoginController : MonoBehaviour
{
    [SerializeField] private InputField accountInput;
    [SerializeField] private InputField passwordInput;

    public void OnLoginClick()
    {
        if (accountInput.text == "lixiaotong" && passwordInput.text == "888888")
        {
            Debug.Log("登陆成功");
            SceneManager.LoadScene("Exam01_ServerScene");
        }
        else
        {
            Debug.Log("登录失败");
        }
    }
}
```

### 📌 原理加餐（A）

> 考纲：**U05** · 域 **5、6** · L2

- **是什么**：`Start` 里一次性 `Instantiate` ×10，parent = **Content**，GridLayout 自动排版。  
- **为什么不用手摆 10 个**：试卷考察 **Prefab + 循环**；手摆改数量要改场景。  
- **单选怎么做**：List 持 `currentSelected`，点击时 **全部 false → 当前 true**；O(10) 可忽略。  
- **Sprite Swap vs 改 Color**：Swap 换资源图是 RPG UI 常规做法；Color  tint 适合简单原型。  
- **SetActive(false) vs Destroy**：关 Panel 可再打开；Destroy 需重建列表（卷面不要求）。
