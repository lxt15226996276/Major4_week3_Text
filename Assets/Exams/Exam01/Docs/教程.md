**易错点**

- 本卷 **无选服场景** — 勿沿用 W2 Exam01 的 `ServerSelectScene`  
- 账号密码 **自定义**（本仓库建议 `const` 固定为 `lixiaotong` / `123456`，卷面允许自定）  
- 背包 **20 条必须代码 Instantiate**，Content 初始 **0 条**；**Grid Constraint Count = 4**  
- 技能 **4 个** + **详情弹窗** 是 **第二层 Panel**，不是只改 Text  
- 主角动作用仓库 **`Assets/Animator/Master.controller`** — 参数 **`Attack` / `SkillU` / `SkillI`**（Trigger），**禁止**写 `IsMoving`  
- 考试要求**新建项目**；本仓库用 `Assets/Exams/Exam01/` 模拟隔离

## §二、场景清单

| 场景文件（建议名） | 用途 | Build Index |
|-------------------|------|:-----------:|
| `Exam01_LoginScene.unity` | 账号密码 + 登录按钮 | **0** |

**Hierarchy 规划（LoginScene）**

```
Exam01_LoginScene
├── Main Camera
├── Directional Light
├── EventSystem
└── LoginCanvas（可复用 LoginUI.prefab · Unpack）
    ├── BgImage
    ├── AccountInput
    ├── PasswordInput
    ├── LoginButton
    └── TipText（P-L1 可选）
```

**Hierarchy 规划（MainScene）**

```
Exam01_MainScene
├── Main Camera
├── Directional Light
├── EventSystem
├── PlayerRoot
│   └── Player（Exam04/Player.prefab 或 Cube+Animator · Master.controller）
└── MainCanvas
    ├── BtnBackpack
    ├── BtnSkill
    ├── BackpackPanel          ← 默认隐藏 · 含 ScrollView + BtnCloseBackpack
    │   └── BackpackScrollView → Viewport → Content（Grid 4 列 · 运行时 20 Item）
    ├── SkillPanel             ← 默认隐藏 · 4 技能钮 + BtnCloseSkill
    └── SkillDetailPanel       ← 默认隐藏 · 详情 Text + BtnCloseDetail
```
