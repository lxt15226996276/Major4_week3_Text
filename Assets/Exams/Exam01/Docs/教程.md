## 二、场景清单

| 场景文件（建议名） | 用途 | Build Index |
|-------------------|------|-------------|
| `LoginScene.unity` | 账号密码 + 登录按钮 | 0 |
| `ServerSelectScene.unity` | 10 个服务器 + 关闭按钮 | 1 |

**Hierarchy 规划（LoginScene）**

```
LoginScene
├── Main Camera
├── Directional Light
└── Canvas (Screen Space - Overlay)
    ├── BgImage（可选）
    ├── TitleText
    ├── AccountInput   (InputField)
    ├── PasswordInput  (InputField, Content Type = Password)
    ├── LoginButton
    └── TipText（可选）
```

**Hierarchy 规划（ServerSelectScene）**

```
ServerSelectScene
├── Main Camera / Light
└── Canvas
    ├── ServerPanel          ← 关闭时隐藏这个
    │   ├── TitleText
    │   ├── ScrollView → Viewport → Content  ← 动态生成 10 条
    │   └── CloseButton（右上角）
    └── (可选) SelectedTipText
```
