### 2.2 `LoginCanvas`（Prefab 根 · 改 Canvas 三组件）

#### Canvas 组件

| 字段 | 设置值 | 含义 |
|------|--------|------|
| Render Mode | **Screen Space - Overlay** | UI 画在屏幕最前 · 不跟 3D 相机 · 登录页常用 |
| Pixel Perfect | **不勾** | 勾选只做像素对齐 · 2D 像素风才开 |
| Sort Order | **0** | 多 Canvas 时越大越靠前 · 本套一个 Canvas 即可 |
| Target Display | **Display 1** | 多显示器时用 · 默认即可 |
| Additional Shader Channels | **默认** | UI 用法线/Tangent 才开 · 现在不用改 |

#### Canvas Scaler 组件

| 字段 | 设置值 | 含义 |
|------|--------|------|
| UI Scale Mode | **Scale With Screen Size** | 按参考分辨率缩放 · 避免窄屏/宽屏 UI 崩 |
| Reference Resolution | **X 1920 · Y 1080** | 设计稿基准 · Game 视图对齐自检 |
| Screen Match Mode | **Match Width Or Height** | 选按宽还是高适配 |
| Match | **0.5** | 0 偏宽 1 偏高 · **0.5 折中** · 登录页常用 |
| Reference Pixels Per Unit | **100** | 与 Sprite/UI 资源一致 · 默认 100 |

#### Graphic Raycaster 组件

| 字段 | 设置值 | 含义 |
|------|--------|------|
| Ignore Reversed Graphics | **勾** | 背面不响应点击 · 常规默认 |
| Blocking Objects | **None** | 是否被 3D 物体挡射线 · Overlay UI 一般 None |
| Blocking Mask | **Everything** | 3D 遮挡层 · 默认即可 |

#### `LoginCanvas` · RectTransform

| 字段 | 设置值 | 含义 |
|------|--------|------|
| Anchor Min / Max | **(0, 0) (0, 0)** | 根 Canvas 随 Prefab 即可 · Scaler 负责缩放 |
| Pivot | **(0.5, 0.5)** | 根节点 pivot 影响小 · 保持默认 |
| Pos X / Y / Z | **0 / 0 / 0** | 根 Canvas 通常为 0 |
| Width / Height | **0 / 0**（或 Scaler 驱动） | Overlay 根常由 Scaler 算 · 勿手改乱 |

### 3.1 AccountData · Dictionary + 会话名

| 步骤 | 操作 |
|------|------|
| 1 | `Scripts/AccountData.cs` · `namespace Exam.Exam07` |
| 2 | 空物体 **`AuthManager`** · 挂 AccountData |
| 3 | **`DontDestroyOnLoad`**（P+1 · 跨场景保留 Dict + 当前用户名） |

#### 本段思路

1. `Dictionary<string,string>` 存 **账号→密码**。  
2. `Register(account, pwd)` · Trim · 判空 · `ContainsKey` 拒重复 · `Add`。  
3. `TryLogin` · `TryGetValue` · 比对密码 · 成功写 **`CurrentUserName`**。  
4. **禁止** UI 脚本里 `new Dictionary`。

#### 带注释 P-L1 范本（AccountData · 节选）

```csharp
namespace Exam.Exam07
{
    /// <summary>
    /// Exam07 账号数据：Dictionary 存注册信息 + 当前登录用户名；跨场景 DDoL。
    /// </summary>
    public class AccountData : MonoBehaviour
    {
        public static AccountData Instance { get; private set; }

        private readonly Dictionary<string, string> accountDic = new Dictionary<string, string>();
        public string CurrentUserName { get; private set; }

        private void Awake()
        {
            if (Instance != null && Instance != this) { Destroy(gameObject); return; }
            Instance = this;
            DontDestroyOnLoad(gameObject);
        }

        /// <summary>思路：Trim → 判空 → 重复键 → Add。</summary>
        public bool Register(string account, string password, out string msg)
        {
            account = account.Trim();
            password = password.Trim();
            if (string.IsNullOrEmpty(account) || string.IsNullOrEmpty(password))
            { msg = "账号或密码不能为空"; return false; }
            if (accountDic.ContainsKey(account))
            { msg = "账号已存在"; return false; }
            accountDic.Add(account, password);
            msg = "注册成功";
            return true;
        }

        /// <summary>思路：TryGetValue 安全查键 → 比密码 → 记录 CurrentUserName。</summary>
        public bool TryLogin(string account, string password, out string msg)
        {
            account = account.Trim();
            password = password.Trim();
            if (!accountDic.TryGetValue(account, out string stored))
            { msg = "账号不存在"; return false; }
            if (stored != password)
            { msg = "密码错误"; return false; }
            CurrentUserName = account;
            msg = "登录成功";
            return true;
        }
    }
}
```
