### 3.0 思路：3D 和 UI 怎么「同时看见」

| 层 | 谁画 | 内容 |
|----|------|------|
| **3D** | `Main Camera` | `Plane` 地面 · `Tank` 主角 ·（下步代码刷）敌人 |
| **UI** | `GameSceneUI` Canvas · **Screen Space - Overlay** | 血条 · 开火钮 · 敌人总血量 · 胜利面板 |

**渲染顺序**：Camera 先画 3D → Canvas 再盖在上面。

**常见挡屏原因**：`GameSceneUI → Backgrouond` 的 **Image 全屏拉伸 + 不透明 Sprite** 会把 3D 整个挡住。  
**正确做法**：**只关 `Backgrouond` 的 Image 组件**（或 Source Image 清空 / Alpha=0），**不要关 `Backgrouond` 物体本身**——它的子物体 `HPPanel` / `BattlePanel` / `GameOverPanel` 还在下面。

**不必改**：Canvas 根节点 Scale=0（有 **CanvasScaler · Scale With Screen Size** 时是 Unity 常见序列化，Play 时 UI 仍正常）。

### 3.2 操作步骤（在 Unity 里自己做）

**A. 改 Prefab（Project 双击 `GameSceneUI`）**

1. 选中 `Backgrouond` → Inspector **Image** → **取消勾选**（物体保持 Active）  
2. `HPSlider` → Value **0** · Min 0 Max 100 · Interactable 关  
3. `BattlePanel`：删 `BtnAttack20`、`BtnAttack50`；`BtnAttack30` 改名 `BtnFire`，子 Text 改「开火」  
4. `GameOverPanel` → 改名 `VictoryPanel`；子 `GameOverText` → `VictoryText`，文案「游戏胜利」；**Active 关**  
5. `HPPanel` 下 **UI → Legacy → Text**，命名 `EnemyTotalHpText`，Text=**0**  
6. 根 `GameSceneUI` 上若有 **Missing Script / BattleController** → Remove Component  
7. **Apply** Prefab

**B. 摆 `Exam03_GameScene`**

### 📌 原理加餐（A）

**A1 · Overlay Canvas 与 3D 不冲突**  
只要 UI 没有全屏 opaque Image，Camera 画的 3D 会在下面，HUD 浮在上面。

**A2 · 为什么关 Image 不关 Backgrouond**  
`HPPanel` / `BattlePanel` / `VictoryPanel` 是 `Backgrouond` 的子物体；关父物体会连 UI 一起消失。

**A3 · Scale=0 的根 Canvas**  
有 CanvasScaler 时由 scaler 驱动显示；**不是** UI 看不见的原因。
